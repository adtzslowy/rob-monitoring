ROB Monitoring is a real-time environmental monitoring dashboard built to track and visualize coastal flood (ROB) risk indicators. The system continuously collects and displays sensor data such as water level, wind speed, air pressure, humidity, temperature, and wind direction.

The dashboard provides live updates, risk classification (SIAGA, AWAS, BAHAYA), interactive trend charts, and detailed sensor history through modal-based visualizations.
